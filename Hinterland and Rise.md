## To install nbextensions in jupyter notebook following each step in that sequence :-

`open anaconda prompt and follow below steps` :-

1) pip install jupyter_contrib_nbextensions

2) pip install jupyter_nbextensions_configurator

3) jupyter contrib nbextension install --user 

4) jupyter nbextensions_configurator enable --user

5) pip install rise

> After installation restart jupyter notebook. On the home page, enable nbextensions and Hinterland. In Rise package, enable scroll = True and change theme of your choice in theme pointer (eg sky, blood, moon, solarized etc)
